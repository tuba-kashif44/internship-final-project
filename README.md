# Network Packet Sniffer (Internship Project)

This project is a Python-based Network Packet Sniffer developed during my Cybersecurity Internship at NCAI, UET Lahore.

## 🛠️ Prerequisites & Setup
1. **Install Python:** Download from [python.org](https://www.python.org/).
2. **Install Npcap:** Required for Windows. Download from [npcap.com](https://npcap.com/).
3. **Install Scapy:** Run `pip install scapy` in your terminal.

## 🚀 How to Run
1. Open terminal in the project folder.
2. Run with Administrator privileges: `python my_project.py`

## 📸 Output Demo
The project captures 10 packets (TCP/UDP/ICMP). See the result in `output.png.png`.
